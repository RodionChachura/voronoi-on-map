## Run locally

```bash
git clone https://github.com/RodionChachura/voronoi-on-map.git  
cd voronoi-on-map  
npm install  
echo 'REACT_APP_MAPBOX_TOKEN=<YOUR_MAPBOX_TOKEN>' >> .env  
npm run  
```
